Terminal-first Git worktree control

Easy worktree management for the terminal.

LazyWorktree is a fast TUI for creating, reviewing, and switching Git worktrees. It combines Vim-style navigation, PR and CI visibility, and deep Git tooling in one focused interface.

Download Releases Read the Docs
  • Git 2.31+Supported baseline
  • Keyboard-firstVim motions + command palette
  • PR and CI awareGitHub and GitLab workflows

Quick start

$ brew tap chmouel/lazyworktree https://github.com/chmouel/lazyworktree
$ brew install lazyworktree --cask
$ cd your-repository
$ lazyworktree

Why teams choose it

Manage your worktree with Ease.

Smart worktree creation

Create from branches, PRs, MRs, or issues with automatic naming and optional file carry-over.

Deep Git tooling

Stage, commit, diff, cherry-pick, synchronise, and open LazyGit without context switching.

CI visibility

Inspect checks, browse logs, and jump straight to job pages when builds fail.

Command palette

Use a VSCode-style palette with MRU ranking to access actions quickly and consistently.

Session aware

Integrate with tmux or zellij so each worktree can have a dedicated terminal context.

Security-minded hooks

Run init and terminate hooks via `.wt` files with TOFU-based trust controls.

Built for momentum

A clean loop from idea to merged branch.

  1. 01Spawn a worktree from your current branch, issue, or open PR/MR.
  2. 02Code with live status, commit inspection, and instant diffs in the same UI.
  3. 03Review CI, open the PR, prune merged branches, and move to the next task.

Install

Get running in less than one minute.

brew tap chmouel/lazyworktree https://github.com/chmouel/lazyworktree
brew install lazyworktree --cask
yay -S lazyworktree-bin
go install github.com/chmouel/lazyworktree/cmd/lazyworktree@latest

Optional tools: gh or glab for forge data, and delta for enhanced diffs.

Bring calm to your busiest repositories.

LazyWorktree keeps each branch isolated, visible, and easy to manage from the keyboard.
