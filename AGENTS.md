## Project

Textual-based TUI for managing Git worktrees. Python 3.12+.

## Structure

- `main.py`: CLI entry point
- `lazyworktree/app.py`: Main TUI application
- `lazyworktree/git_service.py`: Git operations
- `lazyworktree/models.py`: Data models
- `lazyworktree/screens.py`: UI screens
- `tests/`: pytest tests

## Before Finishing

- Run `ruff format` and `ruff check --fix`
- Run tests: `pytest`
- Verify Python 3.12+ compatibility
